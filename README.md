### Getting Started

1. Install dependencies
   - `npm install`
2. Run the dev server
   - `npm run dev`
3. Open the app
   - http://localhost:3000

### Tech
- Next.js 14 (App Router)
- Tailwind CSS 3
- Heroicons

### Customize
- Replace placeholder videos in `components/Hero.tsx` and `components/Banks.tsx` with BLIK! captures.
- Update metadata in `app/layout.tsx`.
- Contact form posts to `/api/contact` (logs by default). Integrate email/CRM as needed.
